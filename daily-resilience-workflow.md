# Daily Resilience Workflow

## Purpose

This daily workflow keeps progress moving during busy school weeks, low-energy days, career-building pressure, and portfolio development.

The goal is not to complete everything. The goal is to create one small proof of progress each day.

## Total Time

30 minutes

## Step 1: Five-Minute Reset

Ask:

- What is my current energy level?
- What is my main pressure point today?
- What is one thing I can realistically move forward?

Choose one:

- Low Energy Day
- Medium Energy Day
- High Energy Day

## Step 2: Ten-Minute Priority Lock

Write today's top three priorities:

1. School priority:
2. Portfolio priority:
3. LinkedIn priority:

Then choose only one task that must happen today.

## Step 3: Ten-Minute Proof Action

Complete one small action from one of these categories.

### School

- Read one section
- Draft one paragraph
- Add one citation
- Review assignment instructions
- Submit one discussion reply
- Clean up one assignment section

### Portfolio

- Create one workflow step
- Improve one GitHub README section
- Add one denial root-cause example
- Update one Google Sheet tracker
- Write one case-study paragraph
- Map one intake, eligibility, authorization, or documentation breakdown

### LinkedIn

- Leave one thoughtful comment
- Send one connection request
- Reply to one message
- Save one useful healthcare operations post
- Draft one post idea
- Add one portfolio proof item to Featured

### Energy Protection

- Drink water
- Take a five-minute walk
- Do a breathing reset
- Step away from scrolling
- Set a bedtime alarm
- Clean one small workspace area

## Step 4: Five-Minute Reflection

Answer:

1. What did I move forward today?
2. What made today harder?
3. What helped me stay steady?
4. What is the next smallest action tomorrow?

## Daily Rule

Small daily proof beats emotional overload.
